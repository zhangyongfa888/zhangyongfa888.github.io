存在的问题

每次进入文章都是固定的背景音乐

问题原因

是否开启toc
toc: false
toc:
  list_number: true 
  music: [1459980094,29717271,30814948,863481092,485612576,31517407,424995317,38592976,427419509,1328382004,1305365717,25706282,416389005,490602642,26494530,1451141290,1434356520,1356367833,1395338980,1390519535,1379517878,1449871979,1441777485]
  urls: [http://www.zhaixiaoou.com/]
  urlsDesc: [zhaixiaoou-小锡兵鸥]

之前是写在配置中，在TOC(右侧级别分类中)显示播放器。

排查过程中发现是由于ejs生成静态文件时候已经产生好随机数并给每一篇文章赋值好了播放地址。

解决办法

1.使用js动态生成iframe以及随机播放地址

2.生成的iframe插入到对应的节点下

在任何地方只要引入ejs就能够自动播放网易外链音乐了~

现在改到了评论上方~

<script type="text/javascript" src="http://res.zhangyongfa888.site/RandomMusic.js"></script>

使用js,方便更改歌曲连接~
